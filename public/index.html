Iniciando com Netlify Edge Functions

Hello e bem vindo ao tutorial!

Este tutorial vai levar você usar através de Edge Functions no Netlify para construir as seguintes URLs:

1. Hello, world

Nós vamos escrever um Edge Function para servir o texto Hello, World do edge function!

/hello

2. Serve um HTML template

Nós vamos configurar o projeto para servir arquivos Template Html estáticos do diretório /public.

/hello-template/a>

3. Add localizaÇão de dados ao HTML template

vamos escrever Edge function para interceptar um template html estático e atualizar ele usando geolocalização de dados na borda(edge)

/hello-template?method=transform
